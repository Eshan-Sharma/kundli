# Kundli & Panchang

A personal Vedic-astrology app — Janma Kundli, all divisional charts (D1–D64), Vimshottari Dasha, Shadbala, Transit, Ashtakoot Milan with dosha analysis, Hindu Panchang with festivals, special yogas, Choghadia, Rahu Kaal, traditional dietary guidance, and travel-direction recommendations.

## Use it

- **Web**: <https://eshan-sharma.github.io/kundli/>
- **Android APK**: download `app-debug.apk` from the [latest release](https://github.com/Eshan-Sharma/kundli/releases/latest). Open on your Android phone and install (you'll need to allow installs from your browser the first time).

Every push to `main` rebuilds the APK with the latest HTML embedded, so the website and APK stay in sync.
